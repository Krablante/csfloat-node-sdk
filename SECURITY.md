# Security Policy

## Supported Versions

Security fixes will target the latest active development branch first.

## Reporting a Vulnerability

Please do not open public issues for undisclosed security vulnerabilities.

Until a dedicated security contact is set, report issues privately to the maintainer through a non-public channel before public disclosure.

## Scope Notes

This project is an unofficial SDK.

Please include:

1. affected method or endpoint
2. reproduction details
3. whether the issue is read-only or mutation-related
4. whether credentials are involved
